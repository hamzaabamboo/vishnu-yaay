# Vishnu Camp Ending Screen

Ending screen for Vishnu camp
With cool flares and button
![Preview](https://preview.ibb.co/iDjCzT/Capture.png)

## Installation

```
  yarn
```

Please include your own font instead of chulafont.css

## Running

```
  yarn start
```

## Deploy

```
  yarn build
  firebase deploy
```

## Dependencies

- react
- react-emotion
- react-confetti
